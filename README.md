# Claude Plugins

A personal agent plugin marketplace for Claude Code.

## Plugins

| Plugin | Description |
|--------|-------------|
| [estonian-groceries-epromo](./estonian-groceries-epromo) | Shop for groceries from ePromo.ee via Claude Code |

## Setup

```
/plugin marketplace add kristjanr/claude-plugins
```

Then install the plugin:

```
/plugin install estonian-groceries-epromo
```
